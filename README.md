# Agente Financiero

## Features
- Comprehensive finance management system.
- Support for multiple currency conversions.
- Automated budget tracking.
- Expense categorization to analyze spending habits.
- Detailed reporting on income, expenses, and savings.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/alan2502/agente-financiero.git
   ```
2. Navigate into the project directory:
   ```bash
   cd agente-financiero
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```

You can access the application at `http://localhost:3000`.

## Deployment
To deploy the application, follow these steps:
1. Build the application:
   ```bash
   npm run build
   ```
2. Host the built application using a server of your choice, such as nginx or serve.

For further customization, refer to the project documentation on the repository.
